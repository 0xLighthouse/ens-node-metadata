# Group

This node represents a logical grouping of multiple child nodes.

## Attributes

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| name | string | No | The name of the group |
| avatar | string | No | A URL to an image used as an avatar or logo |
| description | string | No | A description of the name |
| display | string | No | A canonical display name for the ENS name; this MUST match the ENS name when its case is folded, and clients should ignore this value if it does not (e.g. "ricmoo.eth" could set this to "RicMoo.eth") |
| email | string | No | An e-mail address |
| keywords | string | No | A list of comma-separated keywords, ordered by most significant first; clients that interpresent this field may choose a threshold beyond which to ignore |
| mail | string | No | A physical mailing address |
| notice | string | No | A notice regarding this name |
| location | string | No | A generic location (e.g. "Toronto, Canada") |
| phone | string | No | A phone number as an E.164 string |
| url | string | No | A website URL |
