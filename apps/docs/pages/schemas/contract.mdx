# Contract

Smart contract metadata profile aligned with ENSIP-24: contract text records and compiled metadata resolver profile.

## Attributes

| Key | Name | Type | Required | Description |
| --- | --- | --- | --- | --- |
| alias | alias | string | Yes | Human-readable contract alias (ENSIP-18 alias) |
| description | description | string | No | Short description of the contract purpose (ENSIP-18 description) |
| avatar | avatar | string | No | Avatar URI for the contract profile (ENSIP-12 avatar) |
| url | url | string | No | Primary project or app URL (ENSIP-18 url) |
| category | category | string | No | Contract category (e.g., defi, gaming, dao, utility, proxy, factory) |
| license | license | string | No | Official software license identifier for the source code |
| docs | docs | string | No | Primary documentation URL for developers and users |
| compiled-metadata | compiledMetadata | string | No | URI to compiler-generated contract metadata (Solidity/Vyper/etc.) or resolver-backed compiledMetadata profile |
| audits | audits | json | No | Array of third-party audit reports |
| proxy | proxy | json | No | Proxy descriptor for upgradeable contracts |
