# Contract

Smart contract metadata profile following Enscribe smart contract metadata format.

## Attributes

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| alias | string | No | Human-readable contract alias (ENSIP-18 alias) |
| description | string | No | Short description of the contract purpose (ENSIP-18 description) |
| avatar | string | No | Avatar URI for the contract profile (ENSIP-12 avatar) |
| category | string | No | Contract category (e.g., defi, gaming, dao, utility, proxy, factory) |
| license | string | No | Official software license identifier for the source code |
| docs | string | No | Primary documentation URL for developers and users |
| compiled-metadata | string | No | A URI to a contract metadata file as generated by a Solidity or Vyper compiler |
| audits | json | No | JSON array of third-party audit reports |
