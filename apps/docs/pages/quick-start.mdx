## For Organizations

### Prerequisites

Before proceeding, make sure you have:

1. **ENS Domain**: Own an ENS domain (e.g., `myorg.eth`)
2. **Wallet**: Metamask or compatible Web3 wallet with admin permissions
3. **ETH**: For gas fees on mainnet or testnet

---

### Step 1: Register Your Organization

```typescript
import { OrgRegistryClient } from '@ens-org-registry/client'

const client = new OrgRegistryClient({ 
  provider: window.ethereum,
  network: 'mainnet' // or 'sepolia' for testnet
})

// Register your organization
await client.registerOrganization({
  name: 'myorg.eth',
  profile: {
    alias: 'My Organization',
    description: 'A decentralized autonomous organization',
    avatar: 'ipfs://QmYourAvatarHash',
    website: 'https://myorg.com',
    email: 'contact@myorg.com'
  },
  governance: {
    votingContract: '0x1234...', // Your DAO's voting contract
    proposalThreshold: '1000000000000000000', // 1 token
    votingDelay: 86400 // 1 day
  }
})
```

### Step 2: Set Multi-chain Addresses

```typescript
// Add addresses for different chains
await client.setMultiChainAddresses('myorg.eth', {
  60: '0x1234...', // Ethereum mainnet
  137: '0x5678...', // Polygon
  42161: '0x9abc...', // Arbitrum
  10: '0xdef0...' // Optimism
})
```

### Step 3: Delegate Subdomains

```typescript
// Allow working groups to manage their subdomains
await client.delegateSubdomain(
  'myorg.eth',
  'engineering', 
  '0xEngineeringTeamMultisig...'
)

// Now engineering.myorg.eth can be managed by the engineering team
```

## For Developers

### Installation

```bash
npm install @ens-org-registry/client @ens-org-registry/react
```

### Basic Usage

```typescript
import { useOrgProfile } from '@ens-org-registry/react'

function OrgProfile({ name }: { name: string }) {
  const { profile, loading, error } = useOrgProfile(name)
  
  if (loading) return <div>Loading...</div>
  if (error) return <div>Error: {error.message}</div>
  
  return (
    <div>
      <img src={profile.avatar} alt={profile.alias} />
      <h1>{profile.alias}</h1>
      <p>{profile.description}</p>
      <a href={profile.website}>Visit Website</a>
    </div>
  )
}
```

### Resolving Multi-chain Addresses

```typescript
import { OrgRegistryClient } from '@ens-org-registry/client'

const client = new OrgRegistryClient({ provider })

// Get organization's Polygon address
const polygonAddress = await client.getAddress('myorg.eth', 137)

// Get all addresses
const addresses = await client.getAllAddresses('myorg.eth')
```
