# ENS: Metadata

## Installation

This monorepo contains `apps/` and supporting `packages/`.

| Component | Description |
|-----------|-------------|
| `apps/docs` | Primary project docs built with Vocs ([Docs](apps/docs/README.md)) |
| `apps/interface` | Next.js web UI ([Docs](apps/interface/README.md)) |
| `apps/contracts` | Experimental smart contracts and sandbox |
| `packages/schemas` | Toolchain for publishing JSON schemas to IPFS |
